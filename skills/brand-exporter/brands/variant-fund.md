# Variant Fund

## Colors

| Name | Hex | Role |
|------|-----|------|
| Variant Red | `#DD3737` | Primary accent, CTAs, highlights |
| Dark Red | `#571313` | Hover states, deep accents, section backgrounds |
| Black | `#000000` | Primary text, dark backgrounds |
| White | `#FFFFFF` | Backgrounds, text on dark |
| Cool Gray | `#ABB8C3` | Secondary text, borders, subtle elements |

## Typography

- **Headings**: SF Pro Bold (fallback: Helvetica Neue Bold, Arial Bold)
- **Body**: SF Pro Regular (fallback: Helvetica Neue, Arial)
- **No serif fonts** — modern and clean

## Scale

| Element | Size | Weight |
|---------|------|--------|
| Display / Hero | 42px+ | Bold |
| Section Heading | 36px | Bold |
| Subheading | 20px | Semibold |
| Body | 16px | Regular |
| Caption | 13px | Regular |

## Design Principles

1. Minimalist — generous whitespace, clean lines
2. Bold accents — red used surgically, not decoratively
3. Modern — flat design, no gradients
4. Professional — confidence without flash
5. Rounded CTAs — pill-shaped buttons (`border-radius: 9999px`)

## Logo

- File: `cropped-variant-logo-512.png` (512x512)
- Placement: top-left or centered on title slides
- Clear space: 1x logo height on all sides

## Layout Patterns

- Cards: white with subtle shadow or border
- Sections: alternate white and light gray backgrounds
- Hero: black background, white text, red accent
- Data: simple tables/charts, red for key metrics

## Rules

- Red is accent only, never a background (exception: small badges)
- Max 3 colors in a single view
- Whitespace is a design element
- WCAG AA contrast required
- When in doubt, simpler
